# 💰 工资报表生成工具 v2.2

> 智能化工资管理解决方案 | Python + ttkbootstrap 打造飞书风格界面

---

## 📌 产品简介

工资报表生成工具是一款专为中小企业设计的工资管理桌面应用，采用 **Python 语言** 开发，使用 **ttkbootstrap** 组件库实现飞书风格（Feishu Design）的现代化界面。

**核心能力：**
- 一键生成多银行代发工资报表
- 员工花名册智能管理
- 个税计算与反算
- 数据校验与备份恢复

---

## 🎯 适用场景

| 场景 | 功能 |
|------|------|
| 月度发薪 | 批量生成个税申报表、银行代发文件 |
| 员工管理 | 花名册维护、增删记录追溯 |
| 数据校验 | 身份证/银行卡号自动校验 |
| 个税计算 | 正向计算个税、税后工资反算税前 |

## ✨ 核心功能

### 1️⃣ 报表生成
- **个税版式**：Excel 格式，用于个税申报
- **莱商银行版式**：TXT/GBK 编码
- **济宁银行版式**：TXT/GBK 编码 ⭐
- **农业银行本行版**：CSV 格式
- **农业银行跨行版**：CSV 格式

**工资录入方式：**
```
方式一：手动输入
张三 5000
李四 6000

方式二：Excel 粘贴
直接从 Excel 复制姓名+工资列粘贴

方式三：文件导入
支持 Excel/CSV 批量导入
```

### 2️⃣ 企业管理
- 维护常用企业列表
- 为每个企业配置需要生成的报表类型
- 生成时自动过滤不需要的报表

### 3️⃣ 员工花名册
- 添加/编辑/删除员工
- 智能粘贴识别员工信息
- Excel/CSV 批量导入导出
- 删除员工可恢复（增删记录功能）

**员工信息字段：**
- 姓名（必填）
- 身份证号码
- 手机号码
- 银行卡号（必填）
- 联行号（12位）
- 开户行全称（必填）

### 4️⃣ 联行号查询 ⭐
- 支持模糊查询（如"济宁银行 汶上"）
- 一键复制联行号

### 5️⃣ 个税计算器
- **正向计算**：税前工资 → 个税
- **逆向反算**：税后工资 → 税前工资

### 6️⃣ 数据校验
- 身份证格式校验（18位、校验码）
- 银行卡号校验（Luhn算法）
- 身份证重复检测

### 7️⃣ 备份恢复 ⭐
- 一键备份全套数据
- 一键恢复历史备份
- 便于数据迁移

## 💻 技术栈

```
开发语言：Python 3.8+
GUI 框架：tkinter + ttkbootstrap（飞书风格）
数据处理：pandas + openpyxl
数据库：SQLite3
打包工具：PyInstaller + Inno Setup
```
