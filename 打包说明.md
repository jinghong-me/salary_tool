# 打包说明

## 方法一：一键打包（推荐）

### 1. 安装依赖

确保已安装 Python 3.8+，然后运行：

```bash
pip install pyinstaller pandas openpyxl ttkbootstrap
```

### 2. 安装 Inno Setup

1. 下载 Inno Setup 6：https://jrsoftware.org/isdl.php
2. 运行安装程序，按默认设置安装

### 3. 运行打包脚本

双击运行 `build_installer.bat`，脚本会自动：
- 检查并安装依赖
- 使用 PyInstaller 打包 EXE
- 使用 Inno Setup 创建安装程序

### 4. 获取安装包

打包完成后，安装包位于：
```
installer/工资报表生成工具_v2.1_安装包.exe
```

## 方法二：手动打包

### 步骤 1：打包 EXE

```bash
pyinstaller --noconfirm --onefile --windowed ^
    --name "工资报表生成工具" ^
    --icon "icon.ico" ^
    --add-data "icon.ico;." ^
    salary_tool_feishu.py
```

### 步骤 2：创建安装程序

1. 打开 Inno Setup Compiler
2. 选择 `setup.iss` 文件
3. 点击菜单 `Build` → `Compile`
4. 等待编译完成

## 安装包功能

安装程序 (`工资报表生成工具_v2.1_安装包.exe`) 提供以下功能：

### 安装时
- ✅ 自动检测并提示旧版本
- ✅ 选择安装目录（默认：Program Files）
- ✅ 可选创建桌面快捷方式
- ✅ 可选创建快速启动栏快捷方式
- ✅ 自动创建开始菜单快捷方式
- ✅ 自动创建导出报表目录

### 卸载时
- ✅ 保留导出报表目录中的数据
- ✅ 清理程序文件和快捷方式
- ✅ 确认提示，防止误操作

### 安装后目录结构
```
C:\Program Files (x86)\工资报表生成工具\
├── 工资报表生成工具.exe    # 主程序
├── icon.ico                # 程序图标
├── 员工花名册.xlsx         # 数据文件（首次安装）
├── README.md               # 说明文档
├── requirements.txt        # 依赖列表
└── 导出报表\               # 报表导出目录
    └── ...
```

## 分发说明

打包完成后，可以分发以下文件：

1. **完整安装包**：`installer/工资报表生成工具_v2.1_安装包.exe`
   - 适合普通用户，双击安装即可使用
   - 包含自动创建快捷方式等功能

2. **绿色版**：`dist/工资报表生成工具.exe`
   - 适合高级用户，无需安装
   - 直接运行，但需手动创建快捷方式

## 注意事项

1. **首次运行**：安装后首次运行可能需要几秒钟初始化
2. **数据文件**：员工花名册.xlsx 首次安装时会复制，后续更新不会覆盖
3. **导出目录**：导出报表目录会在首次运行时自动创建
4. **权限问题**：建议以普通用户权限安装，避免写入权限问题

## 常见问题

### Q: 安装时提示缺少 DLL？
A: 确保目标电脑已安装 [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)

### Q: 如何更新版本？
A: 先卸载旧版本，再安装新版本。导出报表目录的数据会自动保留。

### Q: 如何修改默认安装目录？
A: 编辑 `setup.iss` 文件，修改 `DefaultDirName` 行。

### Q: 如何添加更多文件到安装包？
A: 编辑 `setup.iss` 文件，在 `[Files]` 段添加新的 `Source` 行。
